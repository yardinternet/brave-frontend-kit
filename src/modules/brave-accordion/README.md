# BraveAccordion

Wrapper around [accordion-js](https://github.com/michu2k/Accordion) that initialises accordion behaviour on one or more containers. Use in combination with the accordion components from [brave-components](https://github.com/yardinternet/brave-components).

```javascript
import { BraveAccordion } from '@yardinternet/brave-frontend-kit';

new BraveAccordion();
```
